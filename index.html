Trasforma i tuoi dati aperti in RDF semantico

Incolla il tuo CSV, aggiungi i metadati e il sistema genera il file Turtle (TTL) arricchito con le ontologie OntoPiA, allineato a DCAT-AP_IT 3.0 e pronto per dati.gov.it. La tua API key non viene mai salvata né trasmessa a terzi.
